# Citation & Bibliography Manager

Format academic references accurately across all major citation styles.

## Supported Styles

| Style | Primary Use | Key Features |
|-------|------------|--------------|
| Harvard | Regent's default, UK business/social sciences | Author-date in-text, alphabetical reference list |
| APA 7th | Psychology, education, social sciences | Author-date, DOI required, running head |
| Chicago (Notes) | History, humanities, arts | Footnotes/endnotes + bibliography |
| Chicago (Author-Date) | Sciences, social sciences | Similar to Harvard but with Chicago formatting |
| MLA 9th | Literature, languages, cultural studies | Author-page in-text, Works Cited |
| Vancouver | Medicine, biomedical sciences | Numbered sequential citations |
| IEEE | Engineering, computer science | Numbered sequential, bracket notation |

## Regent's University London Default

Regent's uses **Harvard referencing** as the institutional standard. When no style is specified, default to Harvard.

### Harvard Quick Reference

**In-text citation:**
- Single author: (Smith, 2024)
- Two authors: (Smith and Jones, 2024)
- Three+ authors: (Smith et al., 2024)
- Direct quote: (Smith, 2024, p. 45)
- Multiple sources: (Smith, 2024; Jones, 2023)

**Reference list formats:**

Book:
`Surname, Initials. (Year) Title in italics. Edition. Place: Publisher.`

Journal article:
`Surname, Initials. (Year) 'Article title', Journal Name in italics, Volume(Issue), pp. X-Y. doi:xxx`

Website:
`Author/Organization. (Year) Title. Available at: URL (Accessed: Day Month Year).`

Chapter in edited book:
`Surname, Initials. (Year) 'Chapter title', in Editor Surname, Initials. (ed.) Book Title. Place: Publisher, pp. X-Y.`

## Workflow

1. **Identify source type** (book, journal, website, report, etc.)
2. **Extract metadata** (author, year, title, publisher, DOI, URL)
3. **Apply style rules** for the requested citation format
4. **Format in-text citation** and **reference list entry**
5. **Validate** — check for missing fields, correct punctuation, italicization

## Common Tasks

### Convert between styles
Given a citation in one style, reformat to another while preserving all metadata.

### Build reference list
From a set of sources, create a properly formatted, alphabetized (or numbered) reference list.

### Annotated bibliography
For each source: full citation + 150-word annotation covering:
- summary of main argument/findings
- methodology assessment
- relevance to research question

### Check citations
Verify citation accuracy: author names, dates, page numbers, DOI validity.

## Quality Rules
- Never invent citations or fabricate source details
- Flag when metadata is incomplete
- Preserve exact author names and spellings
- Use sentence case for article titles (APA) vs title case (Chicago)
- Always include DOI when available
- Use "Available at:" for URLs in Harvard style
- Use "Accessed:" date for all web sources

Research a topic thoroughly and produce a structured report. The user will specify the topic.

Steps:

1. **Scope**: Clarify what specifically needs to be researched. Define 3-5 key questions to answer.

2. **Search**: Use web search to find authoritative sources. Prioritize official documentation, peer-reviewed content, and reputable technical blogs. Fetch and read at least 5 distinct sources.

3. **Analyze**: Cross-reference findings. Note where sources agree and disagree. Identify gaps in available information.

4. **Synthesize**: Produce a structured report with:
   - Executive summary (3-5 sentences)
   - Key findings (bulleted, with source attribution)
   - Comparison table (if comparing options)
   - Recommendations (ranked by confidence)
   - Sources list (with URLs)
   - Open questions (what remains unclear)

5. **Save**: Write the report to a markdown file in the current project directory.

Mark any uncertain claims with [uncertain]. Never fabricate sources or statistics.
